### Задача 1







### Текст задачи







#### Сформировать список из степеней числа 2.







### Алгоритм решения



######



#### Программа запускает рекурсию «inputNum», внутри которой для получения запроса чисел у пользователя необходимо ввести «ex» как условие выхода. В процессе работы пользователь вводит число либо команду завершения. При вводе числа происходит преобразование строки в целочисленный тип, вычисляется остаток от деления на 10 для получения последней цифры, и запускается новая рекурсия, где эта цифра добавляется в конец списка. Перед началом сбора данных программа через printf запрашивает ввод максимальной степени для числа 2, после чего формирует итоговый результат и выводит его на экран, ожидая нажатия Enter для выхода.

# Задание 2







### Задача 1



###



### Текст задачи







#### Найти первую цифру натурального числа.







### Алгоритм решения







#### Программа запрашивает у пользователя ввод через printf «Введите натуральное число: », после чего считывает строку и проверяет её на корректность. При помощи System.Int32.TryParse происходит попытка преобразования ввода в число: если введена строка или число меньше единицы, выводится сообщение об ошибке. В случае успешного ввода программа сохраняет исходное значение и запускает цикл while, который последовательно делит число на 10 до тех пор, пока оно не станет меньше десяти. В результате этого процесса выделяется первая цифра числа, которая выводится на экран вместе с исходным значением, после чего выполнение останавливается до нажатия Enter.
